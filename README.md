# XAD Monorepo

## Running the Monorepo

You can now use these commands from the root:

```bash
# Run all apps in dev mode
bun run dev

# Run specific app
bun run dev:extension
bun run dev:api

# Build all apps
bun run build

# Run with turbo directly
bunx turbo run dev --filter=@xad/extension
```
